# Custom Dynamic Container

Простая реализация динамического массива (контейнера) на Java, **запрещающая использование встроенных коллекций** (таких как `ArrayList`, `LinkedList` и т.д.).

Проект выполнен в качестве лабораторной работы, демонстрирующего ручную реализацию базовых операций с коллекциями.

## Особенности

* **Generic-класс** (`Container<T>`), позволяющий хранить элементы произвольного типа.
* **Реализация на основе массива** (`Object[]`).
* **Динамическое изменение размера** (удвоение емкости при переполнении).
* Реализованы основные операции: **добавление** (`add`), **извлечение** (`get`), **удаление** (`remove`), **получение размера** (`size`).
* Использование **JavaDoc** для документации публичного API.

## Технологии и Сборка

Проект использует **Apache Maven** для управления зависимостями и сборки.

| Технология | Версия | Назначение |
| :--- | :--- | :--- |
| **Java** | 11+ | Основной язык реализации. |
| **Maven** | 3.x | Управление проектом и сборка. |
| **JUnit 5** | 5.9.1 | Фреймворк для модульного тестирования. |
